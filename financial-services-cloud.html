Financial Services Cloud

O Financial Services Cloud (FSC) é uma expansão do CRM focada na indústria financeira. Ele oferece modelos de dados e processos adaptados a bancos, seguradoras e instituições de investimento.

Nível Iniciante

Modelagem de dados financeiros e noções de relacionamento.

  • Household e PersonAccount: quando usar agrupamento familiar ou conta individual.
  • FinancialAccount e FinancialGoal: tipos (bancário, investimentos, seguros) e ciclos de vida.
  • Guided Engagement e Path para orientar o atendimento.
  • Exercícios: criar household com várias contas, definir meta financeira e configurar Path Guidance.

Nível Intermediário

Compliance e relatórios para insights financeiros.

  • Field‑Level Encryption e Masking para proteção de dados sensíveis.
  • Relatórios e dashboards multi‑objeto para patrimônio líquido e performance de advisors.
  • Ferramentas de produtividade de advisors: workspaces e action plans.
  • Exercícios: aplicar criptografia em campos, criar relatórios Joined e configurar Action Plan.

Nível Avançado

Desenvolvimento customizado e integrações bancárias.

  • Criação de componentes LWC com bibliotecas de gráficos (D3.js) para visualizar dados de investimentos.
  • Batch Apex para conciliação de transações financeiras.
  • Streaming API e Platform Events para captura de eventos em tempo real.
  • Exercícios: desenvolver gráfico de evolução patrimonial com dados externos, implementar batch de reconciliação.

Nível Especialista/Consultor

Governança global, conformidade e continuidade.

  • Estratégias multi‑org versus multi‑divisão para requisitos de data residency.
  • Planejamento de disaster recovery e business continuity (backup, restore, failover).
  • Compliance regulatório (GDPR, Basel) e automatização de backups via APIs.
  • Exercícios: desenhar arquitetura para banco global com várias jurisdições, desenvolver script de exportação periódica.
