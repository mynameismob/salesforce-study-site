Marketing Cloud

O Marketing Cloud Engagement é a plataforma da Salesforce para automação de marketing omnicanal. Abaixo estão os tópicos de estudo divididos por nível.

Nível Iniciante

Fundamentos de email marketing e gerenciamento de dados.

  • Email Studio: criação de emails, templates e personalização com AMPscript básico.
  • Data Extensions: tipos (standard, filtered, random), criação e importação de dados.
  • Send Classifications e Delivery Profiles para controle de remetentes e conformidade.
  • Exercícios: importar dados de CSV, configurar teste A/B de assunto e criar conteúdo dinâmico simples.

Nível Intermediário

Automação de campanhas e jornadas do cliente.

  • Automation Studio: importação de arquivos, atividades de SQL e transferência de dados.
  • Journey Builder: entry events, decisões condicionais, esperas e splits de engajamento.
  • Personalização com AMPscript e uso de Mobile Studio para SMS.
  • Exercícios: construir journey de boas‑vindas com email e SMS, escrever AMPscript para ofertas VIP.

Nível Avançado

Scripting e integrações para campanhas em tempo real.

  • Server Side JavaScript (SSJS) e WSProxy para manipulação de dados programática.
  • APIs REST e SOAP do Marketing Cloud para envios transacionais e gerenciamento de dados.
  • Deliverability: configuração de SPF/DKIM/DMARC, warming de IP e monitoramento de engajamento.
  • Exercícios: escrever script SSJS para atualizar data extensions, configurar triggered send via API.

Nível Especialista/Consultor

Estrategias omni‑channel e AI para escala global.

  • Orquestração multicanal: SMS, Push, Ads e Email via Journey Builder.
  • Einstein Recommendations e Content Selection para personalização de ofertas.
  • Integrações com Sales Cloud via Marketing Cloud Connect e Data Cloud para unificação de perfis.
  • Exercícios: desenhar jornada cross‑channel para milhões de contatos, configurar sincronização com Salesforce.
